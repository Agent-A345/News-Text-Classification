## 📰 News Text Classification

A Natural Language Processing (NLP) project built using Python and Jupyter Notebook that automatically classifies news articles into predefined categories based on their textual content. The project demonstrates text preprocessing, feature extraction, machine learning classification, model evaluation, and news category prediction using a real-world news dataset.

## 🚀 Features
* **Automated News Classification** – Predict the category of news articles from text.
* **Natural Language Processing Pipeline** – Clean and preprocess textual data for machine learning.
* **Text Vectorization** – Convert news articles into numerical feature representations.
* **Multi-Class Classification** – Classify articles across multiple news categories.
* **Feature Engineering for NLP** – Extract meaningful textual features for prediction.
* **Data Cleaning & Preprocessing** – Handle punctuation, stopwords, and text normalization.
* **Model Training & Evaluation** – Train and evaluate machine learning classification models.
* **Performance Metrics Analysis** – Measure accuracy, precision, recall, and F1-score.
* **News Dataset Analytics** – Explore category distributions and textual patterns.
* **Real-World News Dataset** – Practice NLP using thousands of real news articles.

## 🛠️ How to Run
1. **Clone the repository**:
```
git clone https://github.com/Agent-A345/News-Text-Classification.git
```
2. **Navigate to the project folder**:
```
cd News-Text-Classification
```
3. **Install required libraries**
```
pip install pandas numpy matplotlib seaborn jupyter scikit-learn nltk
```
4. **Run the notebook**
```
jupyter notebook
```
5. **Open**
```
text_classification_pipeline.ipynb
```
